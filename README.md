## if - else
![Image](https://github.com/topkoka/python_101_ubuntu/blob/master/pic/flow%20control/if-else.png?raw=true)
#### Statement syntax
```python 
if test_expression:
  statements
else:
  statements
```
####  Statement example
```python
x = 5
if x<10:
  print('Condition is TRUE')
else:
  print('Condition is FALSE')
```
#### Output
```
Condition is True
```
