# Data Types

## Python Numbers
    - int
    - float
    - complex

```python
x = 1    # int
y = 2.8  # float
z = 1j   # complex

print(type(x))
print(type(y))
print(type(z))
```
## Python Casting
    - int()
    - float()
    - str() 
```python     
x = int(2.8)     # x will be 2
y = float("3")   # y will be 3.0
z = str(3.0)     # z will be '3.0'
```
